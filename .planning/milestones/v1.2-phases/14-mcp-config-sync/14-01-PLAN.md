    - SecurityScanner::scan_mcp_config detects sk-ant-api Anthropic keys
    - SecurityScanner::scan_mcp_config detects xoxb- Slack tokens

    - SecurityScanner::looks_like_secret returns true for known patterns, false for ${VAR} refs

  <action>
    1. Add `regex` dependency to `crates/aisync-core/Cargo.toml`.

    2. Create `crates/aisync-core/src/mcp.rs` with:
       - `pub struct McpEngine;`
       - `McpEngine::load(project_root: &Path) -> Result<McpConfig, AisyncError>` -- reads `.ai/mcp.toml`, parses with `toml::from_str`, returns empty McpConfig if file missing.
       - `McpEngine::sanitize_env(mcp: &mut McpConfig)` -- for each server's env, if the value matches SecurityScanner::looks_like_secret AND is not already a `${...}` reference, replace with `${KEY_NAME}`.
       - `McpEngine::generate_mcp_json(mcp_config: &McpConfig) -> Result<String, AisyncError>` -- builds `{"mcpServers": {...}}` JSON using serde_json.     3. Create `crates/aisync-core/src/security.rs` with:
       - `pub struct SecurityScanner;`
       - `pub struct SecurityWarning { pub server_name: String, pub env_key: String, pub pattern_name: String }`
       - Static `SECRET_PATTERNS: LazyLock<Vec<(&str, Regex)>>` with patterns for:
         - AWS Access Key: `AKIA[0-9A-Z]{16}`
         - GitHub Token: `gh[ps]_[A-Za-z0-9_]{36,}`
         - GitHub Fine-grained: `github_pat_[A-Za-z0-9_]{22,}`
         - Slack Token: `xox[bpors]-[0-9A-Za-z-]+`
         - Anthropic API Key: `sk-ant-api\d+-[A-Za-z0-9_-]+`
         - OpenAI API Key: `sk-[A-Za-z0-9]{48,}`
       - `SecurityScanner::looks_like_secret(value: &str) -> bool` -- returns true if value matches any pattern AND is not a `${...}` reference.
       - `SecurityScanner::scan_mcp_config(config: &McpConfig) -> Vec<SecurityWarning>` -- iterates all servers' env values, returns warnings for matches.

    4. Register modules in `crates/aisync-core/src/lib.rs`: add `pub mod mcp;` and `pub mod security;` declarations, plus re-exports `pub use mcp::McpEngine;` and `pub use security::{SecurityScanner, SecurityWarning};`.

    Use `std::sync::LazyLock` (stable since Rust 1.80) for pattern compilation -- no external dependency needed.
