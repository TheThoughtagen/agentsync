    - Neither present -> detected=false

    Detection engine:
    - Empty directory -> empty results vec
    - claude-only fixture -> 1 result (ClaudeCode)
    - multi-tool fixture -> 3 results (ClaudeCode, Cursor, OpenCode)
    - ambiguous fixture -> 1 result (OpenCode with Medium confidence)
